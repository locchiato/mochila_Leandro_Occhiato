Git Fue creado por Linus Torvalds en 2005 para finalizar el desarrollo del kernel de Linux.
Antes utilizaban BitKeeper hasta que se convirtio en un software propietario.

Sus principales objetivos fueron:
Velocidad, diseño sencillo, gran soporte para desarrollo no lineal (miles de ramas paralelas), que sea completamente distribuido y que sea capaz de manejar grandes proyectos eficientemente (velocidad y tamaño de los datos)

El 4 de junio de 2018, Microsoft anunció la compra de GitHub por un importe de 7500 millones de dólares. Cerrándose la adquisición el 26 de octubre de 2018.

Los clientes de la interfaz gráfica de usuario (GUI, por sus siglas en inglés)son herramientas que proporcionan una visualización alternativa para Git. Como por ejemplo: Github Desktop, SourceTree, GitKraken, etc.
